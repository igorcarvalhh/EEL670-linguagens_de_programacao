# Linguagem de programação ML (Meta Language)

## Como rodar ML?

Para compilar um programa ML no terminal basta rodar o comando

```
sml ./folder/fileName.sml
```

## Declarando funções

```
fun funName params = <exp>;
```

```
fun cubo n = n * n * n;
```
